# Podstr — AI-переклад субтитрів

**Переклад субтитрів будь-якою мовою за допомогою AI.**
Розширення для Chrome: знаходить субтитри на відеоплатформах і перекладає їх у реальному часі.

## Що це

Розширення перехоплює субтитри на відеосайтах і перекладає їх через AI-моделі (Claude, Gemini, DeepSeek та ін. через OpenRouter). Працює з англійськими, іспанськими, німецькими, фінськими — будь-якими субтитрами. Перекладає на будь-яку мову.

Якщо хтось вже переклав цей епізод — переклад підтягнеться зі спільного кешу миттєво і безкоштовно.

## Хочу дивитися

1. Встанови з [Chrome Web Store](https://chromewebstore.google.com/detail/iophagcapjpmkcpdjkfndpdakipokeih)
2. Відкрий відео з субтитрами → обери мову в пікері → готово

## Хочу перекладати

### Через OpenRouter API (оплата за токени)

1. Отримай ключ на [openrouter.ai](https://openrouter.ai/)
2. Встав ключ у налаштуваннях розширення
3. Обери модель і мову → переклад запуститься автоматично

> Вартість залежить від моделі: DeepSeek V3 — від $0.01 за серію, Claude Opus — $5–15 за годинний епізод.

### Через Claude CLI (безкоштовно з підпискою Max)

Детальна інструкція: [CONTRIBUTING.md](CONTRIBUTING.md)

## Перевірені платформи

| Платформа | Субтитри | Статус |
|------|------|------|
| **YouTube** | Manual CC (not auto-generated) — primary `/api/timedtext` + transcript-panel fallback | Перевірено |
| **Netflix** | TTML via Cadmium player API | Перевірено |
| **HBO Max** | WebVTT via DASH manifest | Перевірено |
| **BBC iPlayer** | TTML/EBU-TT-D | Перевірено |
| **RaiPlay** | SRT via MAIN-world fetch interceptor | Перевірено |
| **kino.pub** | HLS subtitles (Vidstack player) | Перевірено |
| **RTS Planeta** | Native HTML5 `<track>` | Перевірено |
| **ARTE** | HLS subtitles | Перевірено |
| **Plex** | HLS subtitles | Перевірено |
| **Filmzie** | HLS subtitles | Перевірено |

Інші сайти з HLS / VTT / TTML / нативним `<track>` теж можуть працювати — натисніть **Enable** у popup розширення.

## Можливості

- **Багатомовність** — перекладає з будь-якої мови на будь-яку. Українська за замовчуванням
- **Вибір AI-моделі** — DeepSeek, Gemini, Claude Sonnet/Opus, Llama через OpenRouter
- **Спільний кеш** — один переклав, решта дивляться безкоштовно
- **Вартість перекладу** — видно скільки коштував переклад прямо на відео
- **Підлаштування таймінгу** — `[` / `]` для зсуву ±0.5с
- **Налаштування вигляду** — шрифт, колір, прозорість, позиція

## FAQ

### Чи потрібен API-ключ?

Ні, якщо цей епізод вже хтось переклав (спільний кеш). Для нового перекладу — потрібен ключ OpenRouter або підписка Claude Max.

### Це безкоштовно?

Розширення безкоштовне. Переклади зі спільного кешу — безкоштовні. Платити потрібно лише за переклад нового контенту через OpenRouter API.

### Яка якість перекладу?

Залежить від моделі. Claude Opus — на рівні хорошого фансабу, з гумором, сленгом і контекстом. DeepSeek і Gemini Flash — дешевше, але теж читабельно.
