# Podstr

**Дубляж вбиває акторську гру. Вбудовані субтитри — рівень Google Translate. Ви заслуговуєте на краще.**

Podstr — розширення для Chrome, яке перекладає субтитри за допомогою ШІ — Claude, Gemini, DeepSeek — прямо у браузері. Ви чуєте оригінальні голоси. Ви читаєте субтитри, які справді розуміють контекст, гумор і сленг.

## Як це працює

1. Встановіть розширення з Chrome Web Store
2. Відкрийте відео на YouTube, ARTE або будь-якому підтримуваному сайті
3. Оберіть мову в перемикачі субтитрів над відео
4. Субтитри з'являються поверх відео — перекладені ШІ

Якщо хтось уже переклав цей епізод, він завантажиться зі спільного кешу — миттєво й безкоштовно.

## Чому не просто субтитри платформи

| | Субтитри платформи / Google Translate | Podstr |
|---|---|---|
| **Контекст** | Рядок за рядком, без розуміння діалогу | Пакетний переклад із повним контекстом |
| **Гумор і сленг** | Дослівно, часто невлучно | Розуміє жарти, ідіоми, культурні відсилки |
| **Мовні пари** | Обмежені тим, що пропонує платформа | Будь-яка мова → будь-яка з 30 мов |
| **Контроль якості** | Що дали, те й дивись | Обирайте модель: Claude для якості, Gemini для швидкості, DeepSeek для економії |
| **Подвійні субтитри** | Рідко доступні | Оригінал + переклад на екрані одночасно |
| **Вартість** | Безкоштовно (і це видно) | Від $0.007 за епізод. Є безкоштовні моделі |

## Підтримувані платформи

| Платформа | Субтитри | Статус |
|-----------|----------|--------|
| **YouTube** | Manual CC (не автозгенеровані) | Перевірено |
| **BBC iPlayer** | TTML/EBU-TT-D | Перевірено |
| **ARTE** | HLS subtitles | Перевірено |
| **Plex** | HLS subtitles | Перевірено |
| **Filmzie** | HLS subtitles | Перевірено |
| **Netflix** | — | Може працювати, не перевірено |

Інші сайти з HLS/VTT/TTML субтитрами теж можуть працювати — натисніть **Enable** у спливаючому вікні розширення.

## Швидкий старт

**Просто хочете дивитися** — встановіть із [Chrome Web Store](https://chromewebstore.google.com/detail/iophagcapjpmkcpdjkfndpdakipokeih), відкрийте відео, оберіть мову. Якщо епізод є у спільному кеші, він відтвориться миттєво.

**Хочете перекладати новий контент** — вставте [OpenRouter API key](https://openrouter.ai/keys) у налаштуваннях розширення. Доступні безкоштовні ключі. Оберіть модель, відкрийте відео — переклад почнеться автоматично.

## Можливості

- **30 мов перекладу**, інтерфейс 13 мовами
- **Подвійні субтитри** — оригінал + переклад одночасно
- **Різні моделі ШІ** — Claude Sonnet для якості, Gemini Flash для швидкості, DeepSeek для економії. Є безкоштовні моделі
- **Спільний кеш** — один переклав, усі користуються
- **Вартість перекладу** показана прямо на відео ще до початку
- **Гарячі клавіші** — `[` / `]` зсув таймінгу ±0.5с, `B` перемикання позиції, `\` скидання зсуву
- **Налаштування стилю** — шрифт, колір, прозорість, позиція

## Чесно про обмеження

Модель ШІ бачить лише текст — не відео. Вона не знає, хто говорить — чоловік чи жінка, тому родові форми можуть бути неточними. Не завжди розрізняє «ти» і «Ви». Неологізми може перекласти дослівно.

Це не баги — це природні обмеження текстового перекладу. Живий перекладач, працюючи лише з текстом, зіткнувся б із тими самими проблемами.

**Що вдається добре:** Claude Sonnet дає переклади на рівні хороших фансабів — із гумором, сленгом і контекстом. Для більшості контенту ви забудете, що читаєте субтитри, згенеровані ШІ.
