## Як гэта працуе
1. Усталюйце расшырэнне з Chrome Web Store
2. Адкрыйце відэа на YouTube, ARTE або любым падтрыманым сайце
3. Абярыце мову ў панэлі выбару субтытраў над відэа
4. Субтытры з'яўляюцца паверх відэа — перакладзеныя з дапамогай AI

Калі хтосьці ўжо пераклаў гэты эпізод, ён загружаецца з агульнага кэша — імгненна і бясплатна.

## Чаму не проста платформенныя субтытры
| | Субтытры платформы / Google Translate | Podstr |
|---|---|---|
| **Кантэкст** | Радок за радком, без разумення дыялогу | Пераклад батчамі з поўным кантэкстам |
| **Гумар і сленг** | Літаральна, часта памылкова | Разумее жарты, ідыёмы, культурныя адсылкі |
| **Моўныя пары** | Абмежаваныя тым, што прапануе платформа | Любая мова → любая з 30 моў |
| **Кантроль якасці** | Бярыце, што даюць | Абярыце мадэль: Claude для якасці, Gemini для хуткасці, DeepSeek для эканоміі |
| **Падвойныя субтытры** | Рэдка даступныя | Арыгінал + пераклад на экране адначасова |
| **Кошт** | Бясплатна (і гэта відаць) | Ад $0.007 за эпізод. Бясплатныя мадэлі даступныя |

## Хуткі старт
**Проста хочаце глядзець** — усталюйце з [Chrome Web Store](https://chromewebstore.google.com/detail/iophagcapjpmkcpdjkfndpdakipokeih), адкрыйце відэа, абярыце мову. Калі эпізод ёсць у агульным кэшы, ён прайграецца адразу.
**Хочаце перакласці новы кантэнт** — устаўце [API-ключ OpenRouter](https://openrouter.ai/keys) у наладах расшырэння. Бясплатныя ключы даступныя. Абярыце мадэль, адкрыйце відэа — пераклад пачнецца аўтаматычна.

## Магчымасці
- **30 моў перакладу**, інтэрфейс на 13 мовах
- **Падвойныя субтытры** — арыгінал + пераклад адначасова
- **Некалькі мадэляў AI** — Claude Sonnet для якасці, Gemini Flash для хуткасці, DeepSeek для эканоміі. Бясплатныя мадэлі даступныя
- **Агульны кэш** — адзін пераклаў, усе карыстаюцца
- **Кошт перакладу** паказваецца прама на відэа перад пачаткам
- **Гарачыя клавішы** — `[` / `]` зрух таймінгу ±0.5с, `B` пераключэнне пазіцыі, `\` скід зруху
- **Наладка стылю** — шрыфт, колер, празрыстасць, пазіцыя

## Сумленныя абмежаванні
Мадэль AI бачыць толькі тэкст — не відэа. Яна не ведае, гаворыць мужчына ці жанчына, таму родавыя формы могуць быць памылковымі. Яна не заўсёды адрознівае «вы» ад «ты». Неалагізмы могуць быць перакладзены літаральна.
Гэта не багі — гэта ўласцівыя абмежаванні перакладу толькі па тэксце. Перакладчык-чалавек, працуючы толькі з транскрыптам, сутыкнуўся б з тымі ж праблемамі.
**Што атрымліваецца добра:** Claude Sonnet стварае пераклады, параўнальныя з добрымі фансубамі — гумар, сленг і кантэкст захаваны. Для большасці кантэнту вы забудзеце, што чытаеце субтытры, створаныя AI.
