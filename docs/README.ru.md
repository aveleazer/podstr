# Подстрочник

**Смотрите в оригинале, читайте на своём языке.**
Chrome-расширение: ловит субтитры на видеосайтах, переводит через AI в реальном времени.

[Сайт](https://podstr.cc) · [Chrome Web Store](https://chromewebstore.google.com/detail/iophagcapjpmkcpdjkfndpdakipokeih) · [Telegram](https://t.me/podstrcc)

## Зачем

Включаете сериал в оригинале — а русских субтитров нет. Или есть, но машинный перевод, который не понимает шуток и теряет интонацию. Или есть дубляж, но хочется слышать оригинальные голоса.

Подстрочник перехватывает субтитры и переводит с любого языка на любой через Claude, Gemini, DeepSeek — модели, которые понимают контекст диалога, сленг, юмор. Не идеально — про ограничения ниже. Но на голову выше Google Translate.

Если кто-то уже перевёл этот эпизод — перевод подтянется из общего кеша мгновенно. Бесплатно.

## Начать

1. Установите из [Chrome Web Store](https://chromewebstore.google.com/detail/iophagcapjpmkcpdjkfndpdakipokeih)
2. Откройте видео с субтитрами → выберите язык → готово

Для нового перевода нужен ключ [OpenRouter](https://openrouter.ai/keys). Есть бесплатные модели (Llama 4 Maverick).

| Платформа | Формат | Статус |
|-----------|--------|--------|

Другие сайты с HLS / VTT / SRT / TTML / нативным `<track>` тоже могут работать — нажмите **Включить** в popup расширения.

## Что умеет

**Главное:**
- **Общий кеш** — один перевёл, остальные смотрят бесплатно
- **Выбор модели** — DeepSeek ($0.01/серия), Gemini Flash Lite ($0.02), Claude Sonnet ($0.19)
- **30 языков перевода.** Русский по умолчанию
- **Стоимость** — видно сколько стоил перевод прямо на видео

**Удобства:**
- Тайминг — `[` / `]` сдвиг ±0.5с
- Настройка шрифта, цвета, прозрачности, позиции
- Горячие клавиши — `B` позиция, `\` сброс сдвига

## Как устроено

1. Service worker перехватывает запросы субтитров через `chrome.webRequest`
2. Background скачивает файлы, обходя CORS
3. Батчи по 50 строк → OpenRouter API (ваш ключ, ваша модель)
4. Расширение сжимает результат gzip, сохраняет локально + в общий кеш
5. Content script рисует субтитры синхронно с видео

## Ограничения

AI переводит **только текст** — не видит видео. Поэтому:
- Род может быть неправильным (не знает кто говорит — мужчина или женщина)
- Неологизмы и игра слов — иногда переводит буквально
- Регистр речи (формальный/неформальный) — угадывает по контексту, не всегда верно

Это ограничения формата, не конкретной модели. Человек-переводчик, работая только по тексту без видео, столкнулся бы с тем же.

## API-ключ

Расширение использует [OpenRouter](https://openrouter.ai/). Ключ хранится локально в браузере, никуда не отправляется. Подстрочник бесплатен. Оплачивается только использование моделей через OpenRouter.

Типичная стоимость: $0.005–0.05 за эпизод в зависимости от модели. Бесплатные модели — $0.
