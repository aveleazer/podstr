# Podstr

**Дубляж убивает актёрскую игру. Встроенные субтитры — уровень Google Translate. Вы заслуживаете лучшего.**

Podstr — расширение для Chrome, которое переводит субтитры с помощью AI — Claude, Gemini, DeepSeek — прямо в браузере. Вы слышите оригинальные голоса. Вы читаете субтитры, которые действительно понимают контекст, юмор и сленг.

## Как это работает

1. Установите расширение из Chrome Web Store
2. Откройте видео на YouTube, ARTE или другом поддерживаемом сайте
3. Выберите язык в пикере субтитров над видео
4. Субтитры появляются поверх видео — переведённые с помощью AI

Если кто-то уже перевёл этот эпизод, перевод загрузится из общего кеша — мгновенно и бесплатно.

## Почему не обычные субтитры платформы

| | Субтитры платформы / Google Translate | Podstr |
|---|---|---|
| **Контекст** | Построчно, без понимания диалога | Пакетный перевод с полным контекстом |
| **Юмор и сленг** | Буквально, часто неправильно | Понимает шутки, идиомы, культурные отсылки |
| **Языковые пары** | Ограничено тем, что предлагает платформа | Любой язык → любой из 30 языков |
| **Контроль качества** | Что дали, то и смотришь | Выбирайте модель: Claude для качества, Gemini для скорости, DeepSeek для экономии |
| **Двойные субтитры** | Редко доступны | Оригинал + перевод на экране одновременно |
| **Стоимость** | Бесплатно (и это видно) | От $0.007 за эпизод. Есть бесплатные модели |

## Поддерживаемые платформы

| Платформа | Субтитры | Статус |
|----------|-----------|--------|
| **YouTube** | Ручные CC (не автогенерированные) | Проверено |
| **BBC iPlayer** | TTML/EBU-TT-D | Проверено |
| **ARTE** | HLS-субтитры | Проверено |
| **Plex** | HLS-субтитры | Проверено |
| **Filmzie** | HLS-субтитры | Проверено |
| **Netflix** | — | Может работать, не проверено |

Другие сайты с HLS/VTT/TTML-субтитрами тоже могут работать — нажмите **Enable** в popup расширения.

## Быстрый старт

**Просто хотите посмотреть** — установите из [Chrome Web Store](https://chromewebstore.google.com/detail/iophagcapjpmkcpdjkfndpdakipokeih), откройте видео, выберите язык. Если эпизод есть в общем кеше, он воспроизведётся сразу.

**Хотите переводить новый контент** — вставьте [API key OpenRouter](https://openrouter.ai/keys) в настройки расширения. Доступны бесплатные ключи. Выберите модель, откройте видео — перевод начнётся автоматически.

## Возможности

- **30 языков перевода**, интерфейс на 13 языках
- **Двойные субтитры** — оригинал + перевод одновременно
- **Несколько AI-моделей** — Claude Sonnet для качества, Gemini Flash для скорости, DeepSeek для экономии. Есть бесплатные модели
- **Общий кеш** — один перевёл, остальные смотрят
- **Стоимость перевода** отображается прямо на видео до начала перевода
- **Горячие клавиши** — `[` / `]` сдвиг тайминга ±0.5с, `B` переключение позиции, `\` сброс сдвига
- **Настройка внешнего вида** — шрифт, цвет, прозрачность, позиция

## Честные ограничения

AI-модель видит только текст — не видео. Она не знает, говорит мужчина или женщина, поэтому род может быть неправильным. Не всегда различает «ты» и «вы». Неологизмы могут быть переведены буквально.

Это не баги — это объективные ограничения перевода только по тексту. Живой переводчик, работая только с расшифровкой, столкнётся с теми же проблемами.

**Что получается хорошо:** Claude Sonnet выдаёт перевод, сопоставимый с хорошим фансабом — юмор, сленг и контекст на месте. Для большинства контента вы забудете, что читаете субтитры, сгенерированные AI.
